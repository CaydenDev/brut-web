Getting Started

Installation

There are several ways to include BRUT-UI in your project:

Direct Download

<link rel="stylesheet" href="path/to/brut-ui.min.css">

NPM

npm install brut-ui

CDN

<link rel="stylesheet" href="https://brut.cayden.one/dist/brut-ui.min.css">

Design Principles

Brutalism in Web Design

BRUT-UI is inspired by the Brutalist architectural movement, characterized by:

  • Raw, unrefined aesthetics
  • Honest presentation of materials
  • Functional minimalism
  • Bold, distinctive visual elements

Core Principles

  • Rawness and Honesty: Exposing the nature of web elements
  • Bold Contrasts: Strong visual hierarchy
  • Functional Minimalism: Removing unnecessary decoration
  • Performance-Focused: Lightweight with minimal overhead
  • Accessible Design: High contrast and semantic structure

Design System

Typography

Our typography system uses:

  • Primary Font: Inter - Clean, legible sans-serif
  • Monospace: JetBrains Mono - For code and technical content
  • Limited font weights for clarity
  • Consistent line heights for readability

Colors

Our intentionally minimalist color system includes:

  • Monochromatic scale for primary usage
  • Accent colors used sparingly
  • High contrast for accessibility

Spacing

Built on a base unit of 4px (0.25rem) for consistency

--brut-space-1: 0.25rem  /* 4px */
--brut-space-2: 0.5rem   /* 8px */
--brut-space-4: 1rem     /* 16px */
--brut-space-8: 2rem     /* 32px */

Components

Buttons

Various button styles:

<button class="btn btn-primary">Primary</button>

Cards

Content containers with header, body, and footer:

<div class="card">
  <div class="card-header">Title</div>
  <div class="card-body">Content</div>
</div>

Navigation

Navigation components:

<nav class="nav nav-shadow">
  <div class="nav-item">
    <a class="nav-link">Link</a>
  </div>
</nav>

Forms

Form elements with brutalist styling:

<div class="form-group">
  <label>Label</label>
  <input class="form-input">
</div>

Utilities

Spacing

Margin and padding utilities:

  • m-{size} - Margin
  • p-{size} - Padding
  • mx-{size} - Margin x-axis
  • py-{size} - Padding y-axis

Layout

Layout utilities:

  • container - Container
  • grid - Grid layout
  • flex - Flexbox
  • hidden - Hide element
